Media Aritmética Simple



La media aritmética o promedio es la medida de posición más utilizada y se define como la suma de todos los valores observados dividida por el número total de observaciónes. Por costumbre se le llama simplemente media.

Llamamos con x a la media aritmética o promedio de un conjunto de observaciónes x1, x2, x3, ..., xn proveniente de una muestra y con m (miu) a la media aritmética o promedio de un conjunto de observaciónes X1, X2, X3, ..., XN que constituyen la población.

Ejemplo:

El ejemplo puede resultar más fácil de entender, usaremos los datos 1, 2, 3, 4, 5, 6, 7, 8.
El desarrollo sería así:

x = (1/8)(1 + 2 + 3 + 4 + 5 + 6 + 7 + 8)
x = (1/8)36
x = 4.5
