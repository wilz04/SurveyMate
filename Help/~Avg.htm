Media Aritmética Ponderada



En algunos conjuntos de datos hay observaciones que se repiten dos o más veces. llamamos entonces con x1, x2, x3, ..., xk a los valores distintos de las observaciónes y con f1, f2, f3, ..., fk el número de veces que aparece cada uno de los valores respectivamente.

Ejemplo:

El ejemplo puede resultar más fácil de entender, usaremos los datos 1, 2, 3, 4, 5, 6, 7, 8.
En resumen, lo que haremos sera sumar todos los datos tan solo una vez (sin incluir los repetidos) e irlos multiplicando por la cantidad de veces que aparecen.
El desarrollo sería así:

x = (1/8)(1*1 + 2*1 + 3*1 + 4*1 + 5*1 + 6*1 + 7*1 + 8*1)
x = (1/8)(1 + 2 + 3 + 4 + 5 + 6 + 7 + 8)
x = (1/8)36
x = 4.5
